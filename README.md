# DataEngineerChallenge

## Steps to run the code
### Compile the jar
`sbt clean compile`

### Run tests
`sbt test`

### Make Assembly of the JAR
`sbt assembly`

### Run the code
#### It requires 3 arguments to run 
#### 1. Input Path
#### 2. Output Path
#### 3. deploy-mode
`sbt run <input path> <output path> <deploy-mode>`
